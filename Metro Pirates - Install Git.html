Connecting to Metro Pirates Git

This page will list the steps in order to install the git used by the Metro Pirates

  • Install Git
    1. Goto https://git-scm.com/download/win
    2. Run the file that downloads
    3. Next, Next, Next, ...

  • Create User in Github
    1. Goto https://github.com/
    2. Signup - remember your username & password (*)
    3. Github will send you a verification mail to your inbox - verify it

  • Run Git Gui
    1. Clone Existing repository
    2. Source: https://github.com/glevanon/metro-pirates.git
    3. destination: Somewhere in your computer
    4. Edit > Options > Fill in username (use first name) / email
    5. Click Done (You can ignore the rest of options)

  • First push
    1. Edit file README.md (add / delete whatever you want)
    2. Open Git Gui
    3. Click "Rescan"
    4. Click "Stage Changed"
    5. Fill in something in the "Commit Message"
    6. Click "Commit"
    7. Click "Push"
    8. Enter your github username password (*)
